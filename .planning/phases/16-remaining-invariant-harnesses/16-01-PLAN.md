# 16-01: BurnieCoin Supply Invariant (FUZZ-02)

## Goal
Assert BurnieCoin supply conservation across all call sequences.

## Invariants
1. `coin.supplyIncUncirculated() == coin.totalSupply() + coin.vaultMintAllowance()`
2. `coin.supplyIncUncirculated() >= INITIAL_VAULT_ALLOWANCE` (2M BURNIE -- vault allowance starts at 2M and only grows via escrow)

## Implementation
- New file: `test/fuzz/invariant/CoinSupply.inv.t.sol`
- Inherits DeployProtocol, registers existing GameHandler + VRFHandler + WhaleHandler
- Pure view-level assertions on coin contract state

## Acceptance
- `make invariant-test` passes with no violations
- Both invariants hold across 256 runs x 128 depth
