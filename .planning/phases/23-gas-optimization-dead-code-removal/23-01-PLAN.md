- **GAS-04 (redundant_sload, redundant_external_call):** Storage reads that duplicate an earlier read within the same execution path and could be cached in a local variable. Repeated external calls to the same view function. Note: viaIR=true with optimizer runs=200 already caches some -- focus on cross-function patterns the optimizer CANNOT deduplicate.
