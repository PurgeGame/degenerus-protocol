**2. Face Value Computation at Level 0 Purchase (2.4 ETH)**

A whale bundle bought at level 0 covers levels 1-100 (or 0-99?):
a. Compute ticket face value for each covered level using PriceLookupLib:
   - Levels 1-4 (or starting level): 40 tickets * 0.01 ETH = 0.40 ETH per level * 4 levels = 1.60 ETH
   - Wait -- 40 tickets at a level with price 0.01 ETH: face value = 40 * (0.01/400) = 0.001 ETH? Or 40 tickets means 40 "scaled tickets" where 400 = 1 full ticket?
b. CRITICAL: Determine ticket scaling. From research: 1 "full ticket" = quantity 400. So "40 tickets" might mean 40 scaled (= 0.1 full tickets per level) or 40 full tickets per level.
c. Read the contract to determine: what unit does the whale bundle use for ticket quantity? If it is the scaled quantity (where 400 = 1 full ticket), then 40 scaled = 0.1 full tickets per level.
d. Compute face value per level tier:
   - Each level: tickets_at_level * priceForLevel / 400 (if scaled) or tickets_at_level * priceForLevel (if full)
   - Sum across all 100 covered levels

**3. Face Value at Arbitrary Levels (4 ETH, no level guard)**

Compute the same ticket face value calculation for bundles purchased at:
- Level 10 (covering levels 11-110): prices range from 0.04 to 0.16 ETH
- Level 50 (covering levels 51-150): prices range from 0.08 to 0.24 ETH
- Level 100 (covering levels 101-200): repeating cycle, 0.04-0.24 ETH
- Level 200 (covering levels 201-300): same cycle pattern

Does the ticket face value increase at higher levels? Compare against the fixed 4 ETH price.

**4. Lootbox Value Component**

a. What percentage of the whale bundle price goes to lootbox? Read the contract for the exact BPS.
b. Compute lootbox amount: bundle_price * lootbox_bps / 10000.
c. Apply the lootbox EV model from Phase 3b-03:
   - At neutral activity (100% EV): lootbox value = lootbox amount * 1.00
   - With premium whale bundle boost (+40%): activity score increases, EV multiplier increases
   - At max activity (135% EV): lootbox value = lootbox amount * 1.35
d. Note: lootbox amounts for whale bundles (0.24-0.80 ETH) are well under the 10 ETH cap. The cap is not binding for individual whale bundle lootboxes.

**5. Activity Score Boost Value**

a. Premium whale bundle grants +4,000 BPS (40%) activity score.
b. This increases the lootbox EV multiplier for ALL future lootboxes (from ticket purchases at covered levels), not just the whale bundle lootbox.
c. Compute the marginal value: if the player opens 10 ETH of lootboxes per level across 100 levels, the +40% activity boost increases EV on all of them.
d. But: +40% is part of the total activity score. The marginal benefit depends on the player's other components.
   - If all other components are at 0: 4,000 BPS total. EV = 8000 + (4000 * 2000) / 6000 = 9333 BPS = 93.3%. Sub-neutral. No benefit.
   - If player has 60% base (6,000 BPS from other sources): 6,000 + 4,000 = 10,000 BPS. EV = 10,000 + (4,000 * 3,500) / 24,500 = 10,571 BPS = 105.7%.
   - Marginal benefit: 105.7% - 100% = 5.7% on up to 10 ETH cap = 0.57 ETH per level.

**6. Combined Extraction Model**

For each level tier (0, 10, 50, 100, 200), compute:
```
Total Extraction = Ticket Face Value + Lootbox EV + Activity Boost Value
Total Cost = Bundle Price (2.4 or 4 ETH)
Extraction Ratio = Total Extraction / Total Cost
```

IMPORTANT: Ticket face value is NOT liquid ETH. It is future game participation rights. The tickets are queued for future levels. Their real-world value depends on:
- Whether the game reaches those levels
- Whether the player actually wins anything with those tickets
- The probability of jackpot wins with those tickets

Document the distinction between face value (nominal) and expected liquid value (actual).

**7. F01 Impact Assessment**

With the level guard absent (F01 HIGH):
a. Is there a level at which whale bundle extraction ratio exceeds 100%?
b. If yes: document the specific level, computation, and severity.
c. If no: the lack of level guard is economically benign -- the 4 ETH standard price provides sufficient disincentive.

**8. ECON-06 Verdict**

At levels 0-3 (2.4 ETH) and at arbitrary levels (4 ETH):
- Can any whale bundle + lootbox sequence extract more ETH than deposited?
- Factor in: ticket face value is not liquid, lootbox EV is probabilistic, activity boost has diminishing returns.

ECON-06 Verdict: PASS/FAIL with reasoning. If the extraction ratio exceeds 100% in nominal terms but not in expected liquid terms, document as INFORMATIONAL.
