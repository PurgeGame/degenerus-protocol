**Section 5: Per-Ticket Entropy Derivation (_raritySymbolBatch)**

Document the shared LCG PRNG algorithm used by both _raritySymbolBatch implementations (JM:2127-2221 and MM:462-555):
1. baseKey construction: (lvl << 224) | (queueIdx << 192) | (player << 32) -- cite exact line
2. Per-group seed: seed = (baseKey + groupIdx) ^ entropyWord where groupIdx = ticketIndex >> 4
3. LCG initialization: s = uint64(seed) | 1 (ensure odd for full period)
4. LCG step: s = s * TICKET_LCG_MULT + 1 where TICKET_LCG_MULT = 0x5851F42D4C957F2D (JM:170) = 6364136223846793005 (MM:83)
5. Trait generation: traitId = DegenerusTraitUtils.traitFromWord(s) + (uint8(i & 3) << 6)
6. traitFromWord internals: category = weightedBucket(low 32 bits), sub = weightedBucket(high 32 bits), result = (category << 3) | sub -- cite DegenerusTraitUtils.sol:143-150
7. Final trait ID format: 8-bit [QQ][CCC][SSS] (quadrant from i&3, category, sub-bucket)
