**Section 1: Ticket Purchase (ETH)** -- INFLOW-01
- Entry point: `DegenerusGame.purchase()` -> `DegenerusGameMintModule.purchase()`
- Cost formula: `ticketCost = (price * ticketQuantity) / (4 * TICKET_SCALE)` where TICKET_SCALE=100
- Minimum: TICKET_MIN_BUYIN_WEI = 0.0025 ether
- Payment kinds: DirectEth, Claimable, Combined -- document each path
- Pool split: 10% future (PURCHASE_TO_FUTURE_BPS=1000), 90% next
- Include ticket price tier table from PriceLookupLib (levels 0-4: 0.01 ETH, 5-9: 0.02 ETH, etc.)

**Section 2: Lootbox Purchase (ETH)** -- INFLOW-01
- Entry: Same `purchase()` function, `lootBoxAmount > 0`
- Cost: Direct ETH amount, minimum LOOTBOX_MIN = 0.01 ether
- Three split modes: normal (90/10 future/next), presale (40/40/20 future/next/vault), distress (100% next)
- Document the vault transfer during presale (real ETH transfer, not accounting)

**Section 3: Whale Bundle** -- INFLOW-01
- Entry: `purchaseWhaleBundle(buyer, quantity)` via WhaleModule
- Pricing: WHALE_BUNDLE_EARLY_PRICE=2.4 ether (levels 0-3), WHALE_BUNDLE_STANDARD_PRICE=4 ether (levels 4+)
- Boon discounts: tier 1=1000bps(10%), tier 2=2500bps(25%), tier 3=5000bps(50%)
- Pool split: level 0 = 30/70 next/future, level > 0 = 5/95 next/future
- Lootbox component: 20% presale, 10% post-presale (virtual balance, does NOT reduce pool allocation)

**Section 4: Lazy Pass** -- INFLOW-01
- Entry: `purchaseLazyPass(buyer)` via WhaleModule
- Pricing: levels 0-2 flat 0.24 ETH benefitValue, levels 3+ sum of 10 level prices
- _lazyPassCost formula: sum of PriceLookupLib.priceForLevel(startLevel + i) for i in 0..9
- Boon discount: LAZY_PASS_BOON_DEFAULT_DISCOUNT_BPS=1000 (10%)
- Pool split: 10% future (LAZY_PASS_TO_FUTURE_BPS=1000), 90% next
- Lootbox: based on benefitValue (undiscounted), not totalPrice

**Section 5: Deity Pass** -- INFLOW-01
- Entry: `purchaseDeityPass(buyer, symbolId)` via WhaleModule
- Pricing: basePrice = DEITY_PASS_BASE(24 ETH) + triangular(k) where k=passes sold
- Formula: `basePrice = 24 ether + (k * (k+1) * 1 ether) / 2`
- Boon discounts: same tiers as whale bundle
- Pool split: same as whale bundle (level 0: 30/70, level > 0: 5/95)
- Lootbox: 20% presale, 10% post-presale

**Section 6: BURNIE-to-Ticket Conversion** -- INFLOW-02
- purchaseCoin: `coinCost = (quantity * 250 ether) / 100`, 1 level = 1000 BURNIE
- BURNIE burned, zero ETH pool contribution
- Cutoff: blocked after 90 days (level > 0) or 335 days (level 0)
- purchaseBurnieLootbox: BURNIE burned, min 1000 BURNIE, virtualEth for RNG only
- virtualEth formula: `(burnieAmount * priceWei) / PRICE_COIN_UNIT`
- _ethToBurnieValue for affiliate conversions

**Section 7: Degenerette Wagers** -- INFLOW-03
- Entry: `placeFullTicketBets()` via DegeneretteModule
- Min bets: ETH=0.005, BURNIE=100, WWXRP=1
- ETH routing: 100% to future pool
- Payout cap: ETH_WIN_CAP_BPS=1000 (10% of futurePool)
- BURNIE/WWXRP bets: burned, no pool impact

**Section 8: Presale vs Post-Presale** -- INFLOW-04
- Toggle: `lootboxPresaleActive` starts true
- Auto-end conditions (OR): level >= 3, lootboxPresaleMintEth >= 200 ether
- One-way: can never be re-enabled
- Feature comparison table: lootbox splits, whale/lazy/deity lootbox %, BURNIE rewards, bonusFlip
