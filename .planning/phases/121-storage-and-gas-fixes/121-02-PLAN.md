

    ```solidity
    emit RewardJackpotsSettled(lvl, futurePoolLocal + rebuyDelta, claimableDelta);
    ```

    IMPORTANT: `rebuyDelta` is declared inside the `if (futurePoolLocal != baseFuturePool)` block at L245. It is NOT in scope at L252. Two options:

    Option A (preferred): Hoist `rebuyDelta` declaration to before the if-block, initialize to 0:
    ```solidity
    uint256 rebuyDelta;
    if (futurePoolLocal != baseFuturePool) {
        rebuyDelta = _getFuturePrizePool() - baseFuturePool;
        _setFuturePrizePool(futurePoolLocal + rebuyDelta);

        claimablePool += claimableDelta;
    ```

    When `futurePoolLocal != baseFuturePool`, `rebuyDelta` captures the delta, and the event emits the post-reconciliation value — correct.

    This is a cosmetic fix with no on-chain state impact (per D-09). It only affects event data seen by indexers and frontends.
