```
git stash --include-untracked
```

**Step 1 — Capture baseline storage layouts (parent commit):**
```
git checkout a2d1c585^
```
Then for each of these 7 contracts, run forge inspect and save output:
- `DegenerusGame`
- `DegenerusGameAdvanceModule`
- `DegenerusGameJackpotModule`
- `DegenerusGameDecimatorModule`
- `DegenerusGamePayoutUtils`
- `IDegenerusGame`
- `IDegenerusGameModules`

Command pattern per contract:
```
forge inspect {ContractName} storage-layout --pretty > /tmp/layout-baseline-{ContractName}.txt
```

For the two interfaces (IDegenerusGame, IDegenerusGameModules), expect empty storage layout output (interfaces have no state variables). Still capture the output to document this.

**Step 2 — Capture current storage layouts (HEAD):**
```
git checkout -
```
Then repeat the same `forge inspect` commands, saving to `/tmp/layout-current-{ContractName}.txt`.

**Step 3 — Diff each pair:**
```
diff /tmp/layout-baseline-{ContractName}.txt /tmp/layout-current-{ContractName}.txt
```

If diff produces no output for a contract, that contract's layout is IDENTICAL. If diff produces output, that is a FINDING.

**Step 4 — Restore stash:**
```
git stash pop
```

**Step 5 — Record results:**
Begin writing 191-01-SUMMARY.md with the LAYOUT-01 section. For each contract, record:
- Contract name
- Slot count (from forge inspect output)
- Diff result: "IDENTICAL" or the specific differences
- Overall LAYOUT-01 verdict: PASS (all identical) or FAIL (any differences found)

Per D-01: all 7 targets must be compared.
Per D-02: baseline is parent commit a2d1c585^.
