**System Card 1: PAY-07 — Coinflip Deposit/Win/Loss**

- Trigger: `Player deposits BURNIE into coinflip; resolved via VRF the following day`

- Calculation: `win = (rngWord & 1) == 1` (50/50). Variable multiplier: roll 0 = 1.5x, roll 1 = 2.5x, rolls 2-19 = 1.78x-2.15x. Payout = `stake + (stake * uint256(rewardPercent)) / 100`
- Recipients: `Winning player (BURNIE minted) or house (BURNIE burned on loss)`
- Claim Mechanism: `claimCoinflips() or claimCoinflipsFromBurnie() — both route to _claimCoinflipsInternal`
- Currency: `BURNIE`

Formula block (exact):
```
// BurnieCoinflip.sol:835
bool win = (rngWord & 1) == 1; // 50/50 fair coin

// BurnieCoinflip.sol:813-825 — variable multiplier
roll = seedWord % 20;
// roll == 0:    rewardPercent = 50  (1.5x payout)
// roll == 1:    rewardPercent = 150 (2.5x payout)
// roll 2-19:    rewardPercent = seedWord % 38 + 78 (1.78x-2.15x)

// Payout:
uint256 payout = stake + (stake * uint256(rewardPercent)) / 100;

// Net house edge: ~1.575% (players pay via expected burn > expected mint)
```
