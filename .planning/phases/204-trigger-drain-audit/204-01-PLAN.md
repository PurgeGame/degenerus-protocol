|----------|-------------|
| advanceGame caller -> _handleGameOverPath | External caller (anyone) can trigger gameover via advanceGame; liveness guard is the only gate |
| _handleGameOverPath -> handleGameOverDrain | delegatecall boundary; drain runs in caller's storage context |
| handleGameOverDrain -> runTerminalDecimatorJackpot/runTerminalJackpot | self-call boundary; msg.sender == address(this) guard prevents external invocation |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-204-02 | Denial of Service | _gameOverEntropy | mitigate | Verify 3-day fallback prevents permanent VRF stall from blocking gameover. Audit traces this in TRIG-02, Path C. |
