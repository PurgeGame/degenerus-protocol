### B. MintModule: Replace all coin.notifyQuestMint calls with quests.handleMint

There are 4 call sites. Each `coin.notifyQuestMint(player, qty, paidWithEth)` becomes a call to `quests.handleMint(player, qty, paidWithEth)` that captures the return value and does creditFlip + recordMintQuestStreak.

Define a private helper in MintModule to avoid code duplication at 4 sites:
```solidity
/// @dev Route quest progress directly to DegenerusQuests and handle rewards.
function _questMint(address player, uint32 quantity, bool paidWithEth) private {
    (
        uint256 reward,
        uint8 questType,
        uint32 streak,
        bool completed
    ) = quests.handleMint(player, quantity, paidWithEth);
    if (completed) {
        if (reward != 0) {
            coinflip.creditFlip(player, reward);
        }
        // QUEST_TYPE_MINT_ETH = 1: record mint streak on slot-0 ETH quest completion
        if (paidWithEth && questType == 1) {
            IDegenerusGame(address(this)).recordMintQuestStreak(player);
        }
    }
}
```

Replace each call site:
- Line 804: `coin.notifyQuestMint(buyer, uint32(scaled), true)` -> `_questMint(buyer, uint32(scaled), true)`
- Line 910: `coin.notifyQuestMint(payer, questQty, false)` -> `_questMint(payer, questQty, false)`
- Line 944: `coin.notifyQuestMint(payer, uint32(scaled), true)` -> `_questMint(payer, uint32(scaled), true)`
- Line 1049: `coin.notifyQuestMint(buyer, uint32(questUnitsRaw), false)` -> `_questMint(buyer, uint32(questUnitsRaw), false)`
