### A. Expand onlyCoin modifier (line 293)

Add `ContractAddresses.GAME` as a third accepted address per D-11. This allows game modules (MintModule, DegeneretteModule) to call handlers directly via delegatecall.

Change from:
```solidity
modifier onlyCoin() {
    address sender = msg.sender;
    if (sender != ContractAddresses.COIN && sender != ContractAddresses.COINFLIP) revert OnlyCoin();
    _;
}
```

To:
```solidity
modifier onlyCoin() {
    address sender = msg.sender;
    if (
        sender != ContractAddresses.COIN &&
        sender != ContractAddresses.COINFLIP &&
        sender != ContractAddresses.GAME &&
        sender != ContractAddresses.AFFILIATE
    ) revert OnlyCoin();
    _;
}
```

Update NatSpec from "Restricts access to the authorized COIN or COINFLIP contract" to "Restricts access to authorized COIN, COINFLIP, or GAME contracts."
