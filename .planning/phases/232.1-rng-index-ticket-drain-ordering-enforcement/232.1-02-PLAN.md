2. Call advance() which triggers _swapAndFreeze + LR_INDEX bump from X to X+1 + VRF request
3. Mock VRF fulfillment lands rngWordCurrent (handler.mockVrfFulfill())
4. Call advance() AGAIN -- this is where the bug manifested pre-fix (drain at L242-262 reads lootboxRngWordByIndex[X+1-1]=lootboxRngWordByIndex[X])
