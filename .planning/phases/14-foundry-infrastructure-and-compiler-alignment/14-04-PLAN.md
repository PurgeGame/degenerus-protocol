2. Contract `VRFHandler is Test` with:
   - State: `MockVRFCoordinator public vrf`, `DegenerusGame public game`, `uint256 public ghost_vrfFulfillments`
   - Constructor: takes `(MockVRFCoordinator vrf_, DegenerusGame game_)` and stores them

3. `fulfillVrf(uint256 randomWord) external`:
   - Get `reqId = vrf.lastRequestId()`
   - If `reqId == 0`, return (no requests ever made)
   - Check `(, , bool fulfilled) = vrf.pendingRequests(reqId)` -- if fulfilled, return
   - Try `vrf.fulfillRandomWords(reqId, randomWord)` -- on success increment `ghost_vrfFulfillments`, catch and ignore errors

4. `warpPastVrfTimeout() external`:
   - `vm.warp(block.timestamp + 18 hours + 1)` (matches mainnet VRF retry timeout)

5. `warpTime(uint256 delta) external`:
   - `delta = bound(delta, 1 minutes, 30 days)`
   - `vm.warp(block.timestamp + delta)`

This handler is designed for reuse in Phase 15+ invariant tests. The fuzzer calls `fulfillVrf` with random words, driving the game through VRF-gated transitions.
