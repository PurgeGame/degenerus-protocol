## Tasks

### Task 1: Skeptic Review + Coverage Check
type="auto"

**Skeptic Review:**
For every VULNERABLE and INVESTIGATE finding:
1. Read the cited code independently
2. Trace the execution path
3. Check all preconditions and guards
4. Classify: CONFIRMED / FALSE POSITIVE / DOWNGRADE TO INFO
5. For FALSE POSITIVE: cite the specific line(s) that prevent the attack
6. For CONFIRMED: agree with severity or propose different severity

**Taskmaster Coverage Verification:**
1. Independently scan both contracts for state-changing functions
2. Compare against COVERAGE-CHECKLIST.md
3. Verify every function has a corresponding ATTACK-REPORT.md section
4. Verify call trees are fully expanded
5. Verify storage write maps are complete
6. Verify cached-local-vs-storage checks present
