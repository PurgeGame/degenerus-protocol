    **1. advanceGame Full Path Gas Analysis**

    For every loop in every module:
    - Identify the loop bound (fixed constant vs. state-dependent)
    - If state-dependent: can an attacker grow the bound?
    - Calculate gas per iteration and max iterations
    - Check for nested loops (multiplicative gas)

    **3. Delegatecall Gas Forwarding**
    - Verify gas is properly forwarded through delegatecall
    - Check if any module can consume all forwarded gas and cause the parent to revert
    - Assess if a failing delegatecall is handled or propagated

    **4. External Call Gas**
    For every external call (token transfers, VRF requests, etc.):
    - Check if the callee can consume unbounded gas (e.g., ERC-777 hooks)
    - Verify gas limits on external calls where appropriate
    - Check for returnbomb attacks (large returndata from untrusted callees)

    **5. Storage Access Patterns**
    - Identify cold vs warm SLOAD patterns in hot paths
    - Check for repeated SLOADs that could be cached
    - Assess storage layout impact on gas

    Report any RNG, money, admin, or composition issues found during gas analysis.
