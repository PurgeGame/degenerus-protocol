<!-- Contract constants from DegenerusVault.sol -->
Vault share math:
- Two share tokens: DGVB (BURNIE claims), DGVE (ETH+stETH claims)
- INITIAL_SUPPLY = 1_000_000_000_000 * 1e18 (1T shares)
- REFILL_SUPPLY = 1_000_000_000_000 * 1e18
- Burn coin: coinOut = (coinReserve * sharesBurned) / totalSupply (floor division)
- Burn ETH: claimValue = (ethReserve * sharesBurned) / totalSupply (floor division)
- ETH preferred over stETH: pay ETH first, stETH for remainder
- If supplyBefore == amount burned, mint REFILL_SUPPLY to burner

<!-- Yield distribution BPS from DegenerusGame yield handler -->
Yield distribution: totalYield = balance surplus above tracked pools
- 4600 BPS (46%) -> futurePool
- 2300 BPS (23%) -> VAULT (claimable via DGVE)
- 2300 BPS (23%) -> DGNRS (claimable via DGNRS burn)
- 800 BPS (8%) -> buffer (unextracted, stays in contract)

<!-- DGNRS burn from DegenerusStonk.sol -->
DGNRS burn-to-extract:
- ethOut = ((ethBal + stethBal + claimableEth) * amount) / totalSupply
- burnieOut = ((burnieBal + claimableBurnie) * amount) / totalSupply
- wwxrpOut = (wwxrpBal * amount) / totalSupply
- ETH preferred: pay ETH first, stETH for remainder

<!-- Vault/DGNRS perpetual entries -->
- Both get 16 tickets/level at purchaseLevel+99
- Both have deity-pass-equivalent activity score (+80% BPS via deityPassCount=1)
