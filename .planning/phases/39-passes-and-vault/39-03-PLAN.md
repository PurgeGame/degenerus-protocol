<!-- Contract deity pass mechanics -->
Deity virtual jackpot entries (from DegenerusGame jackpot resolution):
- For each deity pass holder, add floor(2% of matching-symbol trait bucket ticket count) virtual entries, minimum 2
- Applied per jackpot day in the trait bucket that matches the deity's symbol quadrant

Deity boons (from DegenerusGameWhaleModule.sol):
- 3 boons/day per deity (bitmask enforced, 3 slots per day NOT 5)
- Boon types deterministic from deity+day+slot entropy
- Types: coinflip odds boost (500/1000/2500 BPS), lootbox ETH boost (5%/15%/25%), purchase boost, decimator boost, pass discounts, activity stat boosts
- Short expiry: 2-4 game days

Deity game-over refund (from DegenerusGame):
- Flat 20 ETH/pass, levels 0-9 only
- Budget = totalFunds - claimablePool (budget-capped)
- FIFO order via deityPassOwners array
