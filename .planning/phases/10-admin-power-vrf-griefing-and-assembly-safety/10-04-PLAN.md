VRF constants (from DegenerusGameAdvanceModule.sol — confirm actual values by reading source):
- MIN_LINK_FOR_LOOTBOX_RNG = 40 ether (40 LINK) — lootbox RNG minimum subscription balance
- VRF_CALLBACK_GAS_LIMIT — controls VRF gas reimbursement; affects LINK cost per request
- VRF_REQUEST_CONFIRMATIONS — confirmation depth; affects timing not direct cost

VRF subscription model (Chainlink VRF V2.5):
- Subscription is funded by LINK donated to DegenerusAdmin via LINK.transferAndCall → onTokenTransfer → subscription.fund
- Subscription ID is stored in DegenerusAdmin; ADMIN contract is the subscription owner
- LINK cost per VRF request = gas_premium * gas_used * gas_price / LINK_ETH_price + flat_fee
- Typical mainnet VRF V2.5 cost: 0.001 to 0.01 LINK per request (varies with gas price and key hash tier)
- The subscription is NOT controlled by any on-chain player — only ADMIN can fund or cancel it

Drain analysis from RESEARCH.md:
- Daily game RNG: one _requestRng call per advanceGame() that transitions through the RNG stage
- No minimum balance check on daily game RNG — it will attempt and hard-revert if empty
- External attacker CANNOT drain the subscription — they cannot call requestRandomWords on the coordinator as a consumer; only the GAME contract (which is a registered consumer on the subscription) can make requests charged to the subscription
- Realistic drain path: admin never funds → subscription stays empty → game halted

Player-specific grief analysis from RESEARCH.md:
- claimableWinnings[player] is a pull pattern — only the player (or approved operator) can call claimWinnings
- No admin function modifies claimableWinnings[player] entries
- No admin function blacklists or pauses specific addresses
- setLootboxRngThreshold to max → freezes ALL lootbox outcomes, not one player
- wireVrf to bad coordinator → affects ALL pending lootboxes in the current index window (index is shared across players)
- The one indirect vector: admin can force a bad RNG word via wireVrf that adversely affects whichever lootboxes are pending; but this affects an index range, not a named wallet

Mainnet LINK economics (approximate, cite as estimates):
- LINK price: $10-15 at time of writing (2026-03-04)
- Gas price mainnet: 15-50 gwei
- VRF V2.5 cost per fulfillment at 15 gwei, VRF_CALLBACK_GAS_LIMIT = 200,000: ~0.003-0.01 LINK per day
- To drain 40 LINK (lootbox threshold) at 0.01 LINK/day: 4,000 game days → not economically feasible as an attacker strategy
- External attacker cannot trigger VRF requests — only the GAME contract can as a registered consumer
