function _wadPow(uint256 base, uint256 exp) private pure returns (uint256) {
    uint256 result = 1 ether;
    while (exp > 0) {
        if (exp & 1 == 1) { result = (result * base) / 1 ether; }
        base = (base * base) / 1 ether;
        exp >>= 1;
    }
    return result;
}

// Lines 1630-1637: _projectedDrip — one call to _wadPow + arithmetic
function _projectedDrip(uint256 futurePool, uint256 daysRemaining) private pure returns (uint256) {
    if (daysRemaining == 0) return 0;
    uint256 decayN = _wadPow(DECAY_RATE, daysRemaining);
    return (futurePool * (1 ether - decayN)) / 1 ether;
}

// Lines 1642-1659: _evaluateGameOverPossible — 2 SLOADs + 1 mapping read + _projectedDrip + 1 SSTORE
function _evaluateGameOverPossible(uint24 lvl, uint24 purchaseLevel) private {
    if (lvl < 10) { gameOverPossible = false; return; }
    uint256 nextPool = _getNextPrizePool();       // 1 SLOAD (packed slot, likely warm)

    gameOverPossible = _projectedDrip(_getFuturePrizePool(), daysRemaining) < deficit;

    // gameOverPossible = 1 SSTORE (Slot 1, warm — same slot as other bools)
}

**Section 1: _wadPow Gas Profile (per D-09)**

Analyze _wadPow under worst-case: `exp = 120` (maximum daysRemaining = 120 days liveness guard).

Per-iteration gas breakdown:
- Each iteration: 1 comparison (3 gas), 1 bitwise AND (3 gas), conditional branch
- Odd iteration (exp & 1 == 1): 1 MUL (5 gas) + 1 DIV (5 gas) = 10 gas extra
- Every iteration: 1 MUL (5 gas) + 1 DIV (5 gas) + 1 SHR (3 gas) = 13 gas
- Loop overhead: ~5 gas per iteration

120 in binary = 1111000, so 7 bits = 7 iterations, 4 of which have the odd branch.

Worst-case _wadPow gas: 7 iterations * ~25 gas/iteration + overhead = ~200-300 gas.
With `exp = 120` (7 iterations): ~250 gas total.

No overflow risk: `0.9925e18 * 0.9925e18 = ~0.985e36`, divided by 1e18 = ~0.985e18. All intermediate values stay within uint256 range. Maximum intermediate product: `(1e18)^2 = 1e36`, well under `2^256 ~= 1.16e77`.

**Section 2: _projectedDrip Gas Profile**

One call to _wadPow (~250 gas) + 1 SUB (3 gas) + 1 MUL (5 gas) + 1 DIV (5 gas) = ~270 gas total.

Edge case: `daysRemaining == 0` early return costs ~10 gas (comparison + return).

No underflow risk in `1 ether - decayN`: DECAY_RATE = 0.9925e18 < 1e18, so `_wadPow(0.9925e18, n)` always returns a value < 1e18 for any n > 0. Therefore `1 ether - decayN > 0`.
