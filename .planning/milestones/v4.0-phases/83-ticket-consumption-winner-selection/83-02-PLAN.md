**Section 4: Winner Index Computation by Jackpot Type (TCON-03)**

For EACH of the following 8 jackpot types, document a subsection containing:

a) **Entry point** with file:line
b) **RNG word derivation chain**: trace from rawFulfillRandomWords -> rngWordCurrent -> per-jackpot entropy formula. Include every XOR, shift, and entropyStep call with file:line.
c) **Winner index formula**: the exact expression that computes the winner's index into the holder array
d) **Pool source**: which storage structure and key are read
e) **Deity virtual entries**: whether virtualCount is added to effectiveLen (and the formula: max(holders.length / 50, 2))
f) **Max winners**: the constant or computed cap
g) **Key Solidity lines**: quote the 3-5 most important lines with file:line

**8 jackpot types to document:**

1. **Daily ETH Jackpot (Phase 0)**
   - Entry: payDailyJackpot(true, lvl, randWord) -> _processDailyEthChunk
   - Entropy: randWord ^ (uint256(lvl) << 192) then per-bucket entropyStep
   - Winner index: _randTraitTicketWithIndices -> slice % effectiveLen
   - Pool: traitBurnTicket[lvl]

2. **Daily ETH Jackpot (Carryover / Phase 1)**
   - Entry: payDailyJackpot -> carryover path
   - Entropy: randWord ^ (uint256(carryoverSourceLevel) << 192)
   - Winner index: same as above but different source level
   - Pool: traitBurnTicket[carryoverSourceLevel]

3. **Early-Burn (Non-Daily) ETH Jackpot**
   - Entry: payDailyJackpot(false, lvl, randWord) -> _executeJackpot -> _distributeJackpotEth -> _processJackpotBucket
   - Entropy: randWord ^ (uint256(lvl) << 192) then per-bucket entropyStep
   - Winner index: _randTraitTicketWithIndices at JM:1641
   - Pool: traitBurnTicket[lvl]

4. **Daily Coin Jackpot (Near-Future)**
   - Entry: payDailyJackpotCoinAndTickets or payDailyCoinJackpot -> _awardDailyCoinToTraitWinners
   - Entropy: randWord ^ (uint256(lvl) << 192) ^ uint256(COIN_JACKPOT_TAG)
   - Winner index: _randTraitTicketWithIndices at JM:2459
   - Pool: traitBurnTicket[targetLevel] where targetLevel in [lvl, lvl+4]

5. **Far-Future Coin Jackpot**
   - Entry: _awardFarFutureCoinJackpot
   - Entropy: rngWord ^ (uint256(lvl) << 192) ^ uint256(FAR_FUTURE_COIN_TAG) then per-sample entropyStep
   - Winner index: (entropy >> 32) % queue.length -- NO deity virtual entries
   - Pool: ticketQueue[_tqFarFutureKey(candidate)]

6. **Daily Ticket Jackpot**
   - Entry: _distributeTicketJackpot -> _distributeTicketsToBucket
   - Entropy: randWord ^ (uint256(lvl) << 192) then per-bucket entropyStep
   - Winner index: _randTraitTicket at JM:1191
   - Pool: traitBurnTicket[lvl]

7. **Early-Bird Lootbox Jackpot**
   - Entry: _runEarlyBirdLootboxJackpot
   - Entropy: rngWord (raw, no level XOR) then per-winner entropyStep
   - Winner index: _randTraitTicket at JM:833
   - Pool: traitBurnTicket[lvl] where lvl = current_level + 1

8. **DGNRS Final Day Reward**
   - Entry: awardFinalDayDgnrsReward
   - Entropy: rngWord ^ (uint256(lvl) << 192)
   - Winner index: _randTraitTicket at JM:785
   - Pool: traitBurnTicket[lvl]

Also document the BAF Jackpot (in DegenerusJackpots.sol) as a 9th entry, noting it uses VIEW FUNCTIONS on DegenerusGame rather than direct storage reads, and is affected by DSC-02.
