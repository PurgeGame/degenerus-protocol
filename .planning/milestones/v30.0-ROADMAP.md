## Overview

For every function in `contracts/` that consumes a VRF word, prove that from the moment the VRF request is fired, every variable influencing how that word is eventually consumed is frozen — backward (inputs committed at request time) and forward (consumption-site state un-mutable by any actor between request and consumption), exhaustively enumerated per consumer (no sampling).
