   bytes. Verify the slot 0 layout comment (bit allocation map) accurately reflects:

   - No comment still references slot 2 as a live storage slot

   mintPacked_ bits [185-214]: AFFILIATE_BONUS_LEVEL_SHIFT (bits 185-214 for level) and
   AFFILIATE_BONUS_POINTS_SHIFT (bits 185-214 for points, disambiguated by a second field).

   including what is stored, the shift values, and the mask widths.

   Verify any comment describing currentPrizePool's type, max value, or overflow behavior
   accurately reflects the uint128 constraint.

   Verify the numeric value matches the described bit position (e.g., if comment says "bit 42"

   names, descriptions, and return values match actual code.
