Focus areas for WrappedWrappedXRP:

1. **Wrapping and unwrapping mechanics** — Any comment describing how the wrapping/unwrapping
   process works: which token is deposited, what is minted in return, what is burned when
   unwrapping, and the exchange rate or ratio. Verify against actual code logic.

2. **Access control comments** — Comments describing who can call mint, burn, wrap, or unwrap
   functions. Verify against actual `require` checks, modifiers, or role mappings.

3. **Token math and supply comments** — Any comment describing totalSupply, exchange rate,
   decimal handling, or how wrapping affects underlying balances. Verify numeric values and
   formulas match code.

4. **Event NatSpec** — All `@notice` on emitted events. Verify parameter names and descriptions
   match actual event declarations.

5. **General NatSpec** — Every `@notice`, `@param`, `@return`, `@dev` on public/external
   functions. Verify parameter names, descriptions, and return values match actual signatures.

6. **Constructor and initialization comments** — Comments describing what is set at deploy time.
   Verify against the actual constructor.

7. **Inline clarifying comments** — Any `// e.g.`, `// Note:`, `// only`, etc. Verify accuracy.

Focus areas for DegenerusTraitUtils:

1. **Trait encoding scheme comments** — DegenerusTraitUtils encodes NFT traits. Any comment
   describing the encoding format: bit packing layout, field ordering, how trait IDs are
   assigned, or how the encoded value maps to a metadata attribute. Verify against actual code.

2. **Integration point comments** — Any comment describing which contracts call DegenerusTraitUtils
   or which contracts' data it reads. Verify these references are accurate.

3. **Function NatSpec** — Every `@notice`, `@param`, `@return`, `@dev`. Verify parameter
   names, descriptions, and return value descriptions match actual signatures. For functions
   that return encoded packed values, verify the description of the packed format is correct.

4. **Inline comments** — Any `// bit N`, `// mask`, `// offset`, `// e.g.` etc. Verify accuracy.
