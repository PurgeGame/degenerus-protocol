    **action_burn(uint256 actorSeed, uint256 amount):**
    - `calls_burn++`
    - Early return if `game.gameOver()` or `game.rngLocked()`
    - Get `bal = sdgnrs.balanceOf(currentActor)`, early return if 0
    - `amount = bound(amount, 1, bal)`
    - Check 50% cap: read `SLOT_SUPPLY_SNAPSHOT` and `SLOT_PERIOD_BURNED` via `vm.load`. If `currentBurned + amount > snapshot / 2`, clamp `amount` to `snapshot / 2 - currentBurned`. If that's 0, return.
    - `vm.prank(currentActor)`
    - `try sdgnrs.burn(amount) { ghost_totalBurned += amount; } catch {}`

    **action_advanceDay(uint256 randomWord):**
    - `calls_advanceDay++`
    - `vm.warp(block.timestamp + 1 days)` -- cross day boundary
    - `try game.advanceGame() {} catch {}` -- may trigger VRF request
    - Fulfill VRF if pending: check `vrf.lastRequestId()`, if non-zero and not fulfilled, call `vrf.fulfillRandomWords(reqId, randomWord)`
    - `try game.advanceGame() {} catch {}` -- processes VRF result, runs rngGate which resolves period
    - Call `_checkResolvedPeriods()` to update ghost variables for period index, roll bounds

    **action_claim(uint256 actorSeed):**
    - `calls_claim++`
    - Get `ethBefore = currentActor.balance`
    - `vm.prank(currentActor)`
    - `try sdgnrs.claimRedemption() { ghost_claimCount++; ghost_totalEthClaimed += currentActor.balance - ethBefore; } catch {}`
    - After successful claim, attempt re-claim to test no-double-claim:
      ```solidity
      vm.prank(currentActor);
      try sdgnrs.claimRedemption() { ghost_doubleClaim++; } catch {}
      ```

    **action_triggerGameOver():**
    - `calls_triggerGameOver++`
    - `if (game.gameOver()) return;`
    - Warp past liveness timeout: `vm.warp(block.timestamp + 90 days)` (safe overshoot -- liveness guard triggers well before 90 days)
    - `try game.advanceGame() {} catch {}` -- should trigger game-over path
    - Fulfill VRF if pending, then advance again

    **_checkResolvedPeriods() private:**
    - Read `redemptionPeriodIndex` via `vm.load(address(sdgnrs), bytes32(uint256(SLOT_PERIOD_INDEX)))` -- mask to uint48: `uint48(raw & type(uint48).max)`
    - If `currentPeriodIndex < ghost_lastPeriodIndex`, increment `ghost_periodIndexDecreased`
    - If `currentPeriodIndex > ghost_lastPeriodIndex`, this is a new period. Check the PREVIOUS period's roll:
      - Compute mapping slot: `bytes32 slot = keccak256(abi.encode(uint256(ghost_lastPeriodIndex), uint256(8)));`
      - Read raw: `uint256 raw = uint256(vm.load(address(sdgnrs), slot));`
      - Extract roll: `uint16 roll = uint16(raw & 0xFFFF);`
      - If `roll != 0 && (roll < 25 || roll > 175)`, increment `ghost_rollOutOfBounds`
      - Increment `ghost_periodsResolved`
    - Update `ghost_lastPeriodIndex = currentPeriodIndex`

    **Helper: actorCount() returns (uint256):**
    `return actors.length;`

    **Helper: getActor(uint256 i) external view returns (address):**
    `return actors[i];`
