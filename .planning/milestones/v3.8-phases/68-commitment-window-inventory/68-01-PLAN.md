- Trace backward: similar trait-based selection mechanism
- Must identify: all input variables to coin jackpot selection

**Category 4: Lootbox outcome**

**Category 6: Prize pool consolidation**
- Outcome: `_consolidatePrizePools(lvl, rngWord)` and `_applyTimeBasedFutureTake`
- Trace backward: `rngWord`, `currentPrizePool`, `prizePoolsPacked`
