# Plan 114-03: Skeptic Review + Coverage Verification

## Objective
Execute the Skeptic validation on every VULNERABLE and INVESTIGATE finding from the Mad Genius attack report. Simultaneously, execute the Taskmaster coverage verification to ensure 100% function coverage with no gaps.

## Tasks

### Task 1: Skeptic Review
type="auto"

For every VULNERABLE and INVESTIGATE finding in the ATTACK-REPORT.md:
1. Read the cited code independently (do not trust Mad Genius summary)
2. Trace the execution path -- can the claimed scenario actually happen?
3. Check all preconditions, modifiers, and guards
4. Classify each finding:
   - **CONFIRMED** -- attack is valid, assign/adjust severity
   - **FALSE POSITIVE** -- cite the exact guard/condition that prevents it
   - **DOWNGRADE TO INFO** -- theoretically possible but no material impact

Never dismiss without proof. Cite exact line numbers.

Output: `audit/unit-12/SKEPTIC-REVIEW.md`

### Task 2: Taskmaster Coverage Verification
type="auto"

Verify the Mad Genius report against the COVERAGE-CHECKLIST:
1. Every function on the checklist has a corresponding analysis section
2. Every call tree is FULLY expanded (no "...and other helpers")
3. Every storage write in every descendant call is listed
4. The cached-local-vs-storage check is present for every function
5. No function was dismissed as "simple setter" or "standard ERC20"

Produce interrogation questions for any gaps found.

Output: `audit/unit-12/COVERAGE-REVIEW.md` with PASS or FAIL verdict.

**Done criteria:**
- Every VULNERABLE/INVESTIGATE finding has Skeptic verdict
- Coverage verification complete with PASS/FAIL
- No findings dismissed without citing specific preventive code
- Output: `audit/unit-12/SKEPTIC-REVIEW.md` + `audit/unit-12/COVERAGE-REVIEW.md`
