## Tasks

### Task 1: Build Coverage Checklist
type="auto"

Read every function in all three contracts (DegenerusVaultShare, DegenerusVault, WrappedWrappedXRP) line by line. Produce `audit/unit-12/COVERAGE-CHECKLIST.md` containing:

1. **Function inventory table** per contract:
   - Function name, line range, visibility, state-changing (Y/N), category (B/C/D), risk tier (for Cat B)
   - Every function must be listed -- no omissions

2. **Cross-contract call map**: For each function, list every external call it makes (contract, function, line number)

3. **Storage write map**: For each state-changing function, list every storage variable it writes to (directly or via helpers)

4. **Category classification**:
   - Category B: External/public state-changing functions -> full Mad Genius treatment
   - Category C: Private/internal state-changing helpers -> traced through parents
   - Category D: View/pure functions -> no attack analysis needed

5. **Risk tier assignment** for Category B:
   - CRITICAL: burnCoin, burnEth (fund extraction)
   - HIGH: deposit, unwrap, gamePurchaseDeityPassFromBoon (complex ETH flows)
   - MEDIUM: gamePurchase, gameDegeneretteBetEth, wwxrpMint, mintPrize, vaultMintTo, donate, burnForGame
   - LOW: All other proxy functions, ERC20 operations, settings

**Done criteria:**
- Every function in all three contracts listed
- No "similar to above" shortcuts
- Cross-contract calls fully mapped
- Storage writes per function documented
- Output: `audit/unit-12/COVERAGE-CHECKLIST.md`
