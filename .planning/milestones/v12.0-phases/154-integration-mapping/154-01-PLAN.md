    ```
    1. Read levelQuestPlayerState[player] (1 SLOAD)
    2. Check questLevel == level (level-boundary invalidation)

    3. If completed bit is set: skip (already done this level)
    4. Read levelQuestType[level] (1 SLOAD)

    9. Write levelQuestPlayerState[player] (1 SSTORE)
    ```
