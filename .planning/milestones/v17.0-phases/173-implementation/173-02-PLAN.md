```solidity
            // Affiliate bonus (cached in mintPacked_ on level transitions)
            {
                uint256 cachedLevel = (packed >> BitPackingLib.AFFILIATE_BONUS_LEVEL_SHIFT) & BitPackingLib.MASK_24;
                uint256 affPoints;
                if (cachedLevel == uint256(currLevel)) {
                    affPoints = (packed >> BitPackingLib.AFFILIATE_BONUS_POINTS_SHIFT) & BitPackingLib.MASK_6;

            }
```

Key details:
- `packed` is already loaded at line 88 (`uint256 packed = mintPacked_[player]`). The cache read is FREE -- just bit shifts on an already-loaded word.
- `currLevel` is already read at line 94 (`uint24 currLevel = level`). This is the game state variable `level`.
- The cache key is `cachedLevel == currLevel`: if the cached level matches the current game level, the cached points are valid.
- On cache miss (first mint ever, or level has advanced since last mint), falls through to the original `affiliate.affiliateBonusPointsBest` cross-contract call.
- The block scope `{ }` keeps `cachedLevel` and `affPoints` from polluting the outer scope.
- This function is `internal view` -- it does NOT write to storage. It only reads the cache that recordMintData wrote.
- The replacement preserves the `* 100` multiplication to convert points to basis points.
