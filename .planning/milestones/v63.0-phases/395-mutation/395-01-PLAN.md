(`audit/mutation/run-campaign.sh`) used a per-file NARROW oracle (`forge test --match-contract

line is actually executed.

highest-value functions, with no false survivors caused by an oracle that skips the mutated code.

proving (1) the oracle exercises the mutated code and (2) the restore trap keeps `contracts/`
byte-frozen.
