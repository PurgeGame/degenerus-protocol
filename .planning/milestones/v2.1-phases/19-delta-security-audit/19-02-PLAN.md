**DELTA-04: All game->sDGNRS callsites audit**

1. **Pool enum correctness**: The Pool enum value matches the intended pool. Cross-reference against IStakedDegenerusStonk.sol enum: Whale=0, Affiliate=1, Lootbox=2, Reward=3, Earlybird=4. Example: `dgnrs.transferFromPool(Pool.Reward, ...)` at DegenerusGame.sol:482 -- verify it is Pool.Reward (3), not another pool.

2. **Address correctness**: The sDGNRS address used is `ContractAddresses.SDGNRS` (not the old DGNRS address or any other address). Check what variable name is used (`dgnrs`, `sdgnrs`, `dgnrsContract`) and verify each resolves to SDGNRS. 3. **Return value handling**: `transferFromPool` returns `uint256` (actual amount transferred, which may be less than requested if pool is depleted). 4. **Caller authorization**: Each callsite runs via delegatecall from DegenerusGame, so `msg.sender` to sDGNRS is the game contract address. Verify the `onlyGame` modifier on the sDGNRS function matches.

**DELTA-06: Degenerette DGNRS reward math**

Read DegenerusGameDegeneretteModule.sol function `_awardDegeneretteDgnrs` (expected around line 1166-1180 per research). Verify:
- BPS constants: `DEGEN_DGNRS_6_BPS = 400` (4%), `DEGEN_DGNRS_7_BPS = 800` (8%), `DEGEN_DGNRS_8_BPS = 1500` (15%). Grep for each constant declaration and verify values.
- Match count branching: 6->400, 7->800, else->1500 (covers 8). Verify the else clause is only reachable for matchCount == 8 (check the caller to see what values matchCount can take).
- Pool balance: `sdgnrs.poolBalance(Pool.Reward)` -- verify uses Pool.Reward and the `sdgnrs` variable points to SDGNRS
- Capped bet: `cappedBet = betWei > 1 ether ? 1 ether : betWei` -- verify 1 ETH cap
- Reward formula: `(poolBalance * bps * cappedBet) / (10_000 * 1 ether)` -- verify:
  - Numerator: poolBalance * bps * cappedBet
  - Denominator: 10_000 * 1e18
  - Semantics: reward = poolBalance * (bps/10_000) * (cappedBet/1_ether) = pool_percentage * bet_fraction
  - Overflow check: poolBalance (max ~50B * 1e18 = 5e28) * bps (max 1500) * cappedBet (max 1e18) = 5e28 * 1500 * 1e18 = 7.5e49 -- fits in uint256 (max ~1.15e77)
- Zero reward guard: `if (reward == 0) return;`
- Transfer: `sdgnrs.transferFromPool(Pool.Reward, player, reward)` -- verify pool and recipient
