Zod-to-openapi type mapping reference (for the recursive tree walk):

| Zod construct | openapi.yaml counterpart |
| --- | --- |
| `z.string()` | `type: string` |
| `z.number()` | `type: number` |
| `z.coerce.number()` | `type: number` (the coerce is request-side; response emits number) — verify handler doesn't pass through strings |
| `z.number().int()` | `type: integer` |
| `z.boolean()` | `type: boolean` |
| `z.object({...})` | `type: object` with `properties:` |
| `z.array(X)` | `type: array` with `items: X'` |
| `z.nullable(X)` | `nullable: true` on the field |
| `z.optional(X)` (on object field) | NOT in `required:` list |
| `z.enum([...])` | `enum: [...]` |
| `z.literal(X)` | `enum: [X]` single-element, or `const: X` |
| `z.string().regex(/.../)` | `pattern: ...` |
| `z.string().min(N).max(M)` | `minLength: N, maxLength: M` |
| `z.number().min(N).max(M)` | `minimum: N, maximum: M` |
| `z.record(X, Y)` | `type: object` with `additionalProperties: Y'` |
| `z.union([...])` | `oneOf: [...]` |
| `z.discriminatedUnion('k', [...])` | `oneOf: [...]` with `discriminator:` |

Direction rule per D-225-02: Any Zod↔openapi disagreement defaults to `Direction: code->docs` with resolution `RESOLVED-DOC` (patch openapi.yaml to match Zod). Exception: Zod schema has a comment that contradicts what the Zod code does — that's API-03 (comment->code), not API-04, and it belongs in Plan 225-01's catalog, not here.

Severity promotion rules per 225-CONTEXT.md "Severity scheme" (carried from D-224-04):
- Default: INFO
- Promote to LOW when caller-breaking:
  - Handler returns a field type that Zod schema says is `string` but actually is `number` (runtime serialization mismatch)
  - Response field documented in openapi.yaml that the handler never emits (callers expecting it crash)
- Do NOT promote beyond LOW.

Expansion rule per D-225-03: If any sampled endpoint has ≥1 field-level mismatch finding, audit ALL remaining endpoints in the same route file to the same field-level depth. If all 8 sampled endpoints PASS, extrapolate PASS to the remaining 19 endpoints with the exact statement in truths[4].
