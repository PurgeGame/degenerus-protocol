### Path composition rule

For each `fastify.<method>('<relative>', ...)` in route file <name>.ts, the full URL path is:
  <prefix from server.ts for <name>Routes> + <relative>

Example: leaderboards.ts has `fastify.get('/coinflip', ...)` - full path is `/leaderboards/coinflip` - matches openapi.yaml `/leaderboards/coinflip:` and API.md `### GET /leaderboards/coinflip`.

Exception: health.ts uses `fastify.get('/health', ...)` and registers WITHOUT a prefix - full path stays `/health`.
