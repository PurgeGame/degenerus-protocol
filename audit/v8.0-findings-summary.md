## Code Quality Actions Taken During v8.0

1. Dead code removal: `_lootboxBpsToTier` deleted from DegenerusGameStorage.sol
2. Local variable shadowing eliminated: `ticketLevel` inlined in MintModule
3. Redundant parameter removed: `lvl` removed from `_rollWinningTraits` in JackpotModule
4. Unused event declarations deleted: Approval/ApprovalForAll removed from DegenerusDeityPass
5. Event parameter improvement: `LinkEthFeedUpdated` now emits old+new values (1 of 7 NC-11 instances)
6. BurnieCoinflip immutables converted to constants via ContractAddresses
7. Missing events added: DeityPassPurchased, GameOverDrained, FinalSwept, BoonConsumed (4 functions), AdminSwapEthForStEth, AdminStakeEthForStEth
8. DGNRS `transferFrom` now emits `Approval` event on allowance decrement
9. NatSpec additions: 69 across 32 files (Phase 133)
10. GAS-10 immutable candidates: all 10 confirmed as false positives
