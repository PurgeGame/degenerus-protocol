# Parameter Reference

**Degenerus Protocol v1.1 -- Master Constant Lookup**

This document consolidates every named constant, threshold, BPS value, and timing parameter from the Degenerus protocol into a single reference. It is the final v1.1 milestone deliverable, enabling game theory agents to look up any protocol constant by name without searching contract source.

## How to Use

1. **By category:** Constants are organized into sections by type (BPS, ETH/pricing, timing, operational, special).
2. **By name:** The alphabetical cross-reference index in Section 7 maps every constant name to its section anchor.
3. **By subsystem:** Each section is subdivided by game subsystem (pool splits, coinflip, lootbox, etc.).

### Scale Conventions

| Scale | Divisor | Example |
|-------|---------|---------|
| BPS (basis points) | /10,000 | 1000 BPS = 10% |
| Half-BPS | /20,000 | 300 half-BPS = 1.5% |
| PPM (parts per million) | /1,000,000 | 10000 PPM = 1% |

**`ether` suffix note:** Solidity's `ether` keyword multiplies by 10^18. It is used for any 18-decimal token, not just ETH. Constants in BurnieCoin, BurnieCoinflip, quest, and decimator contracts using `ether` typically represent BURNIE, WWXRP, or DGNRS amounts -- not ETH. The "Human" column always specifies the correct token unit.
