## Discrepancies

### DISCREPANCY-01: `futurePrizePoolTotalView()` Misleading "Aggregate" Description

**Function:** `futurePrizePoolTotalView()` (line 2064)
**Line:** 2062-2063
**Current NatSpec:**
```
/// @notice Get the aggregate future pool reserve.
/// @return The futurePrizePool value (ETH wei).
```
**Actual behavior:** Returns `_getFuturePrizePool()` -- identical to `futurePrizePoolView()` at line 2058.
**Issue:** The word "aggregate" implies summing multiple pools or components, but only a single futurePrizePool is returned. This is identical to the sibling view function.

**Severity:** Informational -- no security impact, but could confuse integrators.
**Suggested correction:**

